=== üîç Nmap Usage ===
nmap -sV -sC <ip>                      # Version detection and default scripts
nmap -A <ip>                           # Aggressive scan (OS, version, script, traceroute)
nmap -p- <ip>                          # Scan all 65535 ports
nmap -sU -p 161 <ip>                   # UDP scan (e.g. SNMP)
nmap -Pn <ip>                          # Disable ping, treat host as up
nmap -p 22,80,443 <ip>                 # Scan specific ports
nmap --script vuln <ip>               # Run vulnerability detection scripts
nmap -oA output <ip>                   # Output in all formats (output.nmap, .xml, .gnmap)
nmap --script=http-enum <ip>          # Enumerate HTTP directories
nmap -sS -T4 <ip>                      # Stealth SYN scan with faster timing
